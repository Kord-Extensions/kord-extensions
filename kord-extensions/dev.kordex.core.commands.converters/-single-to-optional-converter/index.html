SingleToOptionalConverter

class SingleToOptionalConverter<T : Any>(val singleConverter: SingleConverter<T>, newSignatureType: Key? = null, newShowTypeInSignature: Boolean? = null, newErrorType: Key? = null, outputError: Boolean = false, var validator: Validator<T?> = null) : OptionalConverter<T>

A special OptionalConverter that wraps a SingleConverter, effectively turning it into an optional converter with the same logic.

The behaviours specified in OptionalConverter also apply to this converter, so it's worth reading about it.

Parameters

singleConverter

The SingleConverter to wrap.

newSignatureType

An optional signature type string to override the one set in singleConverter.

newShowTypeInSignature

An optional boolean to override the showTypeInSignature setting set in singleConverter.

newErrorTypeString

An optional error type string to override the one set in singleConverter.
