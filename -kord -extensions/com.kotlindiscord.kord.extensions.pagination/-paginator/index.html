Paginator for, well, pagination.

This paginator is fairly extensible, supporting subclassing of this class, as well as the Pages and Page classes. It's designed with page groups in mind, which means you can provide groups of pages that can be switched between using a dedicated reaction.

Parameters

bot

The bot object this paginator was created for

keepEmbed

Whether to keep the embed after the paginator is destroyed, false by default

owner

Optional paginator owner, if you want to prevent other users from using the reactions

pages

Set of pages this paginator should paginate

switchEmoji

If you have multiple groups, this is the emoji used to switch between them

targetChannel

The channel this paginator should be created within

targetMessage

The message this paginator should be created in response to

timeout

Optional timeout, after which the paginator will be destroyed
