Represents the difference between two Kord User objects.

This is intended for use with events that change things, to make logging easier - but may have other applications.

Parameters

avatar

The new object's avatar URL, or null if there's no difference.

discriminator

The new object's discriminator, or null if there's no difference.

flags

The new object's user flags, or null if there's no difference.

username

The new object's username, or null if there's no difference.
