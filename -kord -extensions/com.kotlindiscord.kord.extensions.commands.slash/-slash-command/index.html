open class SlashCommand<T : Arguments>(extension: Extension, arguments: () -> T?, parentCommand: SlashCommand<out Arguments>?, parentGroup: SlashGroup?) : Command
Content copied to clipboard

Class representing a slash command.

You shouldn't need to use this class directly - instead, create an Extension and use the slash command function to register your command, by overriding the Extension.setup function.

Parameters

arguments

Arguments object builder for this command, if it has arguments.

extension

The Extension that registered this command.

parentCommand

If this is a subcommand, the root command this command belongs to.

parentGroup

If this is a grouped subcommand, the group this command belongs to.
