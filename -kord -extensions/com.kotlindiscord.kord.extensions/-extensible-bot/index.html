ExtensibleBot

open class ExtensibleBot(token: String, prefix: String, addHelpExtension: Boolean, addSentryExtension: Boolean, invokeCommandOnMention: Boolean, messageCacheSize: Int, commandThreads: Int, guildsToFill: List<Snowflake>?, fillPresences: Boolean?)

An extensible bot, wrapping a Kord instance.

This is your jumping-off point. ExtensibleBot provides a system for managing extensions, commands and event handlers. Either subclass ExtensibleBot or use it as-is if it suits your needs.

Parameters

addHelpExtension

Whether to automatically install the bundled help command extension.

addSentryExtension

Whether to automatically install the bundled Sentry command extension.

commandThreads

Number of threads to use for command execution. Defaults to twice the number of CPU threads.

fillPresences

Whether to request presences from the members retrieved by guildsToFill.

guildsToFill

Guild IDs to request all members for on connect. Set to null for all guilds, omit for none.

invokeCommandOnMention

Whether to invoke commands that are prefixed with a mention, as well as the prefix.

messageCacheSize

How many previous messages to store - default to 10,000.

prefix

The command prefix, for command invocations on Discord.

token

The Discord bot's login token.
